# eniso_acm_sc_website

## Hidden Features (For Developers)

This project contains a few intentionally hidden or placeholder areas that are not yet ready for public use.

### 🔒 Hidden / Disabled Pages
- **Resources** (`/resources`) is currently hidden from the main navigation and has been simplified to a placeholder page. It will be rebuilt with real content (workshops, projects, guides, etc.) in the future.

### 📝 Notes
- The resources page route still exists so the team can add content later, but it is not linked anywhere in the UI to prevent users from landing on incomplete placeholders.
